WeatherFox

Mar 10th, 2013 | Comments

I’ve been working on a simple Weather application using HTML5-based technologies. I completed the first version recently. The source is available at Github.

The technologies used are:

  • KnockoutJS
  • Yeoman.io
  • RequireJS
  • SCSS
  • jQuery Mobile
  • jQuery deferreds
  • HTML5 GeoLocation API
  • Yahoo! Weather API via Yahoo! Query Language and Yahoo! Pipes
    • The Yahoo! Pipe URL is http://pipes.yahoo.com/pipes/pipe.info?_id=687279a3ea30ea78c978b1bf6e235055
